Q&A

  1. Three main differences between web 1.0, web 2.0 and web 3.0:
    • In web 1.0, there is only limited interaction between the sites and the web users.
    • In web 2.0, unlike web1, there is much more interaction between web users and sites, which allows users to interact more freely with each other.
    • In web 3.0, computers can interpret information like humans and generate and distribute content in an intelligent way for the needs of the users.
  2. A use case that shows the difference between web1, web2 and web3:
    Wordpress is a web 2.0 site, as it allows the users to both search for blogs and to comment on posts of other users' blogs. Users can comment on posts, which is why Wordpress is not a web 1.0 site. Wordpress is not a web 3.0 site because it does not use any intelligence to personalize the blogs for each user.
